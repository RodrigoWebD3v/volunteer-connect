# Fluxos E2E planejados

## Guest

1. Abrir `/`.
2. Listar oportunidades.
3. Abrir detalhe.
4. Tentar se inscrever e ser direcionado para login/cadastro.

## Voluntario

1. Criar conta.
2. Confirmar email.
3. Entrar.
4. Abrir oportunidade publicada.
5. Criar inscricao.
6. Ver `Minhas inscricoes`.
7. Ver `Minhas presencas` apos registro pela ONG.

## ONG

1. Criar conta com CNPJ e logo.
2. Ver estado de analise pendente.
3. Apos aprovacao, criar oportunidade.
4. Ver inscricoes da oportunidade.
5. Registrar presenca apos conclusao.

## Admin

1. Entrar com conta admin seedada.
2. Listar ONGs pendentes.
3. Aprovar ou reprovar com motivo.

## Seguranca nos E2E

Usar apenas fixtures sem dados reais. Nao commitar tokens, emails pessoais, CPF/CNPJ reais ou service role.
