# Frontend

Frontend SvelteKit do Volunteer Connect.

## Pré-requisitos

- Node.js 20+
- npm 10+

## Instalação

```powershell
npm install
```

## Desenvolvimento

```powershell
npm run dev
```

Por padrão, o Vite expõe a aplicação em `http://localhost:5173`.

## Qualidade

Execute estes comandos antes de enviar mudanças do frontend:

```powershell
npm run check
npm run lint
npm run format
npm test
```

Esta base cobre apenas a fundação do frontend. Telas de autenticação, ONG, oportunidades, candidaturas e eventos entram nas próximas fases.
